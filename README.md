# Chromatic

Chromatic is a curated collection of beautiful color palettes designed for designers and developers. Explore 100+ stunning color palettes to find the perfect match for your next project.

## Features

- **Curated Selection**: 100+ hand-picked palettes.
- **Categorized**: Easily browse by themes like Dark, Light, Pastel, Vintage, and more.
- **Copy to Clipboard**: One-click hex code copying.
- **Custom Palettes**: Create and save your own color combinations.
- **Favorites**: Save your favorite palettes for quick access.

## Getting Started

### Prerequisites

- Node.js & npm installed.

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```cd Chromatic```

3. Install dependencies:
   ```npm install```

4. Start the development server:
   ```npm run dev```
